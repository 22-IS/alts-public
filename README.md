# alts-public
Шаблон репозитория студента.

## Статичные файлы

`info.txt` - информация о лабораторной работе.

Формат:
```
<номер_лр>
<код_студента>
```

Пример:
```
4
88a3b0f7afb7d18e70842fd5cdd5f49f
```

## Динамические файлы

`result.txt` - информация о результате проверки. Генерируется после успешной проверки работы. Внутри одно число от 0 до 100 - баллы за выполненную работу.

Формат:
```
<баллы>
```

Пример:
```
100
```
